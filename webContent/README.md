# 🚀 Java Servlet Authentication Project

## 📌 Description
This project is a Java Servlet-based web application that demonstrates the use of core servlet concepts such as **RequestDispatcher**, **forward()**, and **include()** methods.

The application includes basic authentication features like user registration, login, and a dashboard page after successful login.

---

## ✨ Features
- 🔐 User Registration
- 🔑 Login Authentication
- 📊 Dashboard Page
- 🔁 RequestDispatcher usage
- ➡️ forward() method implementation
- ➕ include() method implementation

---

## 🛠️ Technologies Used
- Java (Servlets)
- HTML / JSP
- Apache Tomcat Server

---

## 🧠 Concepts Covered
- RequestDispatcher.forward()
- RequestDispatcher.include()
- Basic Authentication Flow
- Servlet Lifecycle (basic understanding)
- MVC Architecture (basic level)
